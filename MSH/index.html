Welcome to the Magic Square Helper

A unique tool for magicians: create stunning magic squares based on numbers or dates with just one click.

Quick Start Guide

  • Turn on device: Press the button at the bottom using a pen (you'll hear a soft click).
  • Menu navigation: Use Button B to navigate, Button A to select.
  • Enter sum/date: Input your desired values in the START menu.
  • Show magic square: Double-click Button B to calculate and display the square.
  • Additional info: Long press Button B to show additions.
  • Deep sleep: Long press Button B in the main menu to enter sleep mode.
  • Reset: Long press Button A in the main menu.

Firmware Update

You can update the firmware online in just a few steps:

  1. Open the marker and remove the ink cartridge
  2. Locate the USB-C connector inside
  3. Connect it to your computer using a USB-C cable
  4. Open the web updater and follow the instructions
